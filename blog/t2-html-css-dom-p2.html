Terminology pt2

What is grid based design?

Grid based design lets the developer envision the the html page in grid form which makes understanding the layout easy. Making changs to your layout is easier if the layout itself is easy to interpret.

What's all the hype about responsive web pages?

Depending upon the browser or display device used by the user the webpage needs to adjust accordingly. A responsive webpage's contents will not get distorted if the viewer shrinks the page. A responsive webpage retains its structure irrespective of the display device.

What is semantic structure?

Semantic structure determines the prominence of the content in a webpage. It allows the user to arrange important content first and then less important content after it and so on and so forth.

Points worth mentioning

Defining the layout does require a fair amount of guessing. It's best to draw it out on a piece of paper and try to replicate it from there.
